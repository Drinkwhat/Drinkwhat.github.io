app.database module

Database configuration and session management module.

This module handles: - SQLAlchemy engine creation and configuration - Session factory management - Dependency injection for database sessions in FastAPI routes - Support for both PostgreSQL and SQLite databases

app.database.get_engine_and_session()[source]

Create SQLAlchemy engine and session factory based on DATABASE_URL.

Automatically configures: - SQLite: Sets check_same_thread=False for development - PostgreSQL: Uses standard connection pool settings

Returns:

Motore SQLAlchemy e factory della sessione.

Return type:

tuple[Engine, sessionmaker[Session]]

Importa esplicitamente tutti i modelli SQLAlchemy dell’applicazione.

Questo è utile per garantire che Base.metadata includa tutte le tabelle (es. per migrazioni Alembic, introspezione e documentazione).

app.main module

Restrackit FastAPI Application - Main Entry Point.

This is the core FastAPI application that: - Configures CORS and security middleware - Sets up global exception handlers - Registers API routers - Provides health check endpoint

Note: Database schema management is handled externally via Alembic migrations, not automatically at startup.

app.main.lifespan(_)[source]

Gestisce il ciclo di vita dell’applicazione FastAPI.

async app.main.add_process_time_header(request, call_next)[source]

Aggiunge header di tracing/diagnostica ad ogni risposta HTTP.

  • X-Request-ID: usa quello ricevuto (se valido) oppure genera un UUID.

  • X-Process-Time: tempo di processamento (secondi) per la singola richiesta.

Parameters:

  • request (Request) – Richiesta FastAPI in ingresso.

  • call_next (Callable[[Request], Awaitable[Response]]) – Handler downstream che produce la Response.

Returns:

La risposta con header aggiunti.

Return type:

Response
