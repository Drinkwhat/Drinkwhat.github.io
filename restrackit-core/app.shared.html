app.shared.allergen_enum module

class app.shared.allergen_enum.Allergen(*values)[source]

Bases: IntFlag

14 allergeni obbligatori secondo il Reg. UE 1169/2011.

Ogni valore è una potenza di 2 per supportare la rappresentazione bitmask. La combinazione di allergeni si ottiene con l’operatore OR (|).

Esempio:

mask = Allergen.GLUTEN | Allergen.MILK # prodotto con glutine e latte gluten_free = not (mask & Allergen.GLUTEN)

GLUTEN = 1
CRUSTACEANS = 2
EGGS = 4
FISH = 8
PEANUTS = 16
SOY = 32
MILK = 64
TREE_NUTS = 128
CELERY = 256
MUSTARD = 512
SESAME = 1024
SULPHITES = 2048
LUPIN = 4096
MOLLUSCS = 8192
ALL = 16383

app.shared.allergens module

Gestione allergeni secondo il Regolamento UE 1169/2011 (14 allergeni obbligatori).

Utilizza una bitmask per rappresentare in modo compatto tutti gli allergeni in un singolo campo intero del database.

Converte una lista di slug allergeni in una bitmask intera.

Parameters:

allergen_names (list[str]) – Lista di slug allergeni (es. ["glutine", "latte"]).

Returns:

Return type:

int
