app.main module

Restrackit FastAPI Application - Main Entry Point.

This is the core FastAPI application that: - Configures CORS and security middleware - Sets up global exception handlers - Registers API routers - Provides health check endpoint

Note: Database schema management is handled externally via Alembic migrations, not automatically at startup.
